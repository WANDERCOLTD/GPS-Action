  1. Two views, push/pop nav — no 3-pane on phone. Tap a row → push to detail; swipe back to list.

  1. Funnel headline: Delivered → Opened → Clicked → Acted. The "Acted" tile is the activist-meaningful number — verified shares / RSVPs / volunteer signups.
  2. Replies surface as a count — clicking jumps to the originating team's inbox (the cross-cutting seam Q29). Reply lands as a Request threaded against this Broadcast.
  3. Per-segment breakdown — important when the audience spans multiple Groups / Networks. Lets the sender see who responded to what.
  4. Failures expandable with reason codes. Quiet-hours and channel-fallback are operational metrics for tuning.
  5. Privacy default — aggregate metrics for everyone; per-recipient drill-down sysadmin-only Q30. Avoids the "did Sharon read it yet?" surveillance feel.
