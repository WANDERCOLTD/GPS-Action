05 · Share to WhatsApp

The action that spreads the network

Every public-shareable post has a one-tap Share to WhatsApp affordance. User picks recipients in WhatsApp itself; we stay out of the way.

How it works

Tap → WhatsApp opens → pick groups or people → send

Opens WhatsApp pre-filled with a short intro, the post title, and the public URL. WhatsApp auto-generates a preview card using Open Graph tags, so the recipient sees a hero image and context — not a bare link.

Uses the whatsapp://send URL scheme on phone, wa.me universal link as fallback. We can't pre-select recipients (privacy), but we can track that a share was tapped for engagement analytics.
