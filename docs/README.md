# GPS Action Documentation

This directory holds every document in the project.

After running `./import-docs.sh` with the source files in `docs-staging/`,
this will include:

- `feature-spec/v0.5.md` — canonical feature spec
- `architecture/` — ERD, API contract, decision log
- `design-system/` — tokens, components, guide
- `process/` — session briefs, reviewer checklist, ratchet, security, change absorption
- `product/` — scenarios, parking lot
- `adrs/` — architecture decision records

See `index.md` (imported from staging) for the full map.
